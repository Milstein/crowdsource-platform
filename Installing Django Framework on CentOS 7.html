Installing Django Framework on CentOS 7

March 14, 2015March 14, 2015 devadmin Linux, Web Development

Hello Guys , today I will be focusing on Installing Django Framework on CentOS 7.  First of all let me give you a brief introduction of Django. Django is a very powerful web framework for developing web applications based on Python. Using frameworks such as Django allows you to run your sites and applications quicker without worrying about the basic structure of code.

In this guide I will be demonstrating few methods of Installing Django Framework on CentOS 7 server.

Installing the EPEL Repository

The methods we will be discussing now are based on EPEL repos for CentOS . EPEL repository contains extra packages which are not maintained as the core distribution packages.

It is easy to configure access to EPEL repository. You can configure yum so that it can access EPEL repository. by typing


yum install epel-release

After execution of above command  you will have access to all the applications which are maintained by EPEL repository.

 

Installing Django

There are many methods of installing django , every method has its own advantage depending on your requirements. We will be discussing two methods of Installing Django on CentOS 7.

Global Install from Packages

Django Packages can be found in the EPEL repository we just installing and can also be installed using yum package manager. This is the most simple method of installing django but this might not be so flexible as the installed version of django might lack  behind.

So for Installing Django through this method you simple need to execute this command.

yum install python-django

This will install Django on your server and to test the installing and checking the version simply type.


django-admin --version


1.6.10

Output of the above command means that Django is successfully installed on your server but the installed version might not be latest and stable.

 

Global Install Through PIP

PIP is also a package manager but for python. Once pip is installed on your server you can easily install most latest and stable version on any user on server. PIP is the best option for globally installing Django on your server.

You can install pip through the EPEL repository by executing.

sudo yum install python-pip

Once pip is installed you can globally install Django by executing.

sudo pip install django

You can verify the installation version again by executing the previous command.

django-admin --version


1.7.5
